---
title: "Insert HTML"
api: "POST https://app.twilix.io/api/v1/collection/insert-html"
---

Insert HTML contents into your collection.

This endpoint will automatically parse the html for relevant information and stores them in the vectorestore.

If you need to insert data via url directly, see our [Insert URL](/api-reference/endpoint/insertUrl) endpoint.

<Snippet file="param-collection-insert.mdx" />

<ParamField body="html">The raw HTML string to insert.</ParamField>

<ParamField body="metadata" type="object" default="{}">
  Optional metadata when inserting the HTML

<Note>
  Note that this requires an object. For example, an example metadata would be:
  ```
  {
    "name": 'my_html.html',
    "url": 'https://www.example.com'
  }
  ```
</Note>
</ParamField>
