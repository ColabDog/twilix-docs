# Intro

You can use suggest message in our TypeScript SDK to power important use cases such as: 

- Responding to an email thread
- Drafting cold messages or emails
- Crafting personalised invitations or responses

<Snippet file="installation.mdx"/>

```typescript
// Start a conversation
const emailObject = {
       emailThreadID: 123,
       emailID: "email-1",
       author: "rahul",
       content: "Hi David, checking in how are you?",
       responseEmailID: null,
       attachments: null
}

const emailObject2 = {
       emailThreadID: 123,
       emailID: "email-2",
       author: "jacky",
       content: "I'm great thanks! Let's catch up for coffee when you are free.",
       responseEmailID: "email-1",
       attachments: ["www.s3.objects.com/GET"]
}

fetch("https://api.twilix.io/v1/collection/bulkInsert", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Authorization: "Bearer <token>",
  },
  body: JSON.stringify({
    collection: "emailConversation",
    objects: [emailObject, emailObject2],
  }),
})
  .then((response) => response.json())
  .then((data) => {
    // Handle the response data
    console.log(data);
  })
  .catch((error) => {
    // Handle any errors
    console.error("Error:", error);
  });

fetch("https://api.twilix.io/v1/collection/suggestMessage", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Authorization: "Bearer <token>",
  },
  body: JSON.stringify({
    collection: "emailConversation",
  }),
})
  .then((response) => response.json())
  .then((data) => {
    // Handle the response data
    console.log(data);
  })
  .catch((error) => {
    // Handle any errors
    console.error("Error:", error);
  });

```
