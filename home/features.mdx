There are a couple of concepts to know about Twilix.
- Collection refers to a store of objects (similar to a table of rows in SQL). These are case-senstiive!
- Object is any JSON-like storage (similar to a row in SQL)
- Field is the name of each column in your CSV or each key in your JSON
